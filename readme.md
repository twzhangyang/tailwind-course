## Setup css course
1. mkdir tailwind-course
2. cd tailwind-course
3. npm init -y
4. code tailwind-course
5. npm install live-server -D
6. Add the below command into package.json
```
"start": "live-server",
```
7. npm run dev 

## Setup git
1. git init
2. touch .gitignore
3. ignore below files
```
.idea/
node_modules/
```
